# 🎒 BackpackPlugin — Personal Backpacks for Players

**BackpackPlugin** is a simple and convenient plugin for Minecraft servers that adds a virtual backpack to each player. Perfect for survival and RPG-style servers, it provides a clean, persistent, and personal inventory system.

---

## 🔧 Features

- 📦 **`/backpack` Command** – Opens the player’s personal backpack.
- 💾 **Automatic Saving** – Backpack contents are saved between log-ins and server restarts.
- 🎨 **Custom Interface** – The GUI title is styled as `§6Рюкзак` for a better immersive experience.
- 🔒 **Player Isolation** – Each player gets a unique inventory only they can access.
- 📂 **Persistent Storage** – Backpacks are saved to individual `.yml` files in the plugin folder.

---

## 🧪 Commands

```bash
/backpack    # Opens your personal backpack
````

---

## 🛠 Requirements

* **Minecraft**: 1.13 or higher
* **Server**: Paper / Spigot / Bukkit
* **Java**: 8 or higher


## 📁 Data Storage

Each player’s backpack is saved in its own YAML file named like `UUID.yml` inside the plugin’s data folder.

✅ Safe to restart your server — data will not be lost.

---

## 🧩 How to Install

1. Download the plugin `.jar` file.
2. Drop it into your server’s `plugins/` folder.
3. Restart the server.
4. Use `/backpack` in-game!
